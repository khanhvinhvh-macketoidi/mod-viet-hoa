# P5.1 — Production Ready Foundation

Bản này không thay đổi layout Author Center. Nội dung chính:

- metadata động cho từng trang mod;
- canonical, Open Graph và Twitter Card cho mod detail;
- loading UI cho toàn site, thư viện mod và hồ sơ tác giả;
- chuẩn hóa `APP_URL`, tránh URL có hai dấu `/`;
- preflight kiểm tra biến môi trường, file dữ liệu và quyền ghi;
- lệnh kiểm tra phát hành `npm run release:check`.

## Biến môi trường bắt buộc

```env
APP_URL=https://ten-domain-cua-ban.vn
JWT_SECRET=chuoi-ngau-nhien-toi-thieu-32-ky-tu
```

Không dùng dấu `/` ở cuối `APP_URL` cũng được; hệ thống tự chuẩn hóa.

## Kiểm tra trước khi public

```powershell
npm run preflight:production
npm run lint
npm run build
npm run start
```

Hoặc chạy gộp:

```powershell
npm run release:check
```

## Lưu ý triển khai

Website đang lưu dữ liệu trong `data/*.json` và file tải lên trong `public/uploads`. Vì vậy máy chủ production phải có ổ đĩa lưu bền và quyền ghi cho hai vị trí này. Không nên deploy theo mô hình serverless có filesystem tạm thời nếu chưa chuyển dữ liệu và uploads sang database/object storage.
